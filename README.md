# Customer Behavior Data Analysis

This project analyzes retail customer shopping behavior to uncover patterns that can improve sales, customer satisfaction, and long-term loyalty. It follows an end-to-end analytics workflow covering data preparation, exploratory analysis, SQL-based business reporting, and interactive dashboard creation.

## Project Overview
A retail company wants to understand how demographics, product categories, discount behavior, shipping preferences, and subscription status influence purchasing decisions and repeat purchases. This project uses a customer behavior dataset to answer business-focused questions such as:

- Which customer segments generate the most revenue?
- How do discounts affect purchase behavior?
- Which products perform best by review rating and purchase frequency?
- Do subscribers spend more than non-subscribers?
- What factors drive repeat purchases and loyalty?

## Objectives
- Clean and prepare raw customer data for analysis
- Perform exploratory data analysis using Python
- Answer business-focused questions with SQL
- Create visual insights using Power BI
- Translate findings into recommendations for marketing and product strategy

## Tools and Technologies
- Python for data cleaning and analysis
- SQL for customer behavior analysis
- Power BI for dashboard and visualization
- Jupyter Notebook for exploratory analysis

## Files in This Repository
- Customer_Shopping_Behavior_Analysis.ipynb — Python notebook for data cleaning and analysis
- customer_behavior_sql_queries.sql — SQL queries for business analysis
- customer_shopping_behavior.csv — Source dataset
- README.md — Project overview and instructions

## Workflow
1. Import and explore the dataset
2. Clean and transform the data
3. Load the data into a SQL database
4. Run SQL queries to answer business questions
5. Create dashboards and summarize insights
